# Personal Portfolio Website

## Project Overview

This repository contains a simple personal portfolio created for Week 1 (HTML) assignment. The page demonstrates semantic HTML, a contact form, internal navigation, and use of images.

## Files

- `index.html` — Main portfolio page (About, Skills, Contact)
- `Readme.md` — This file (project documentation)
- `images/` — Place your `profile.jpg` image here
- `requirements.txt` — Optional

## How to preview

1. Open `index.html` in your browser.
2. Place a profile image named `profile.jpg` inside the `images/` folder.

## Notes

- The contact form currently uses `action="#"` — replace with a backend endpoint or an email handler to process submissions.
- Validate the HTML using the W3C validator to ensure full marks.
